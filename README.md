# miniguia-estudos-notebooklm
Me chamo Laíla, sou bacharel em Direito, com especialização em andamento em Compliance e Ciência de Dados.
Este é o meu primeiro Projeto a ser entregue para a plataforma DIO, no Bootcamp Bradesco - GenIA, Dados e Cyber.
Neste projeto, desenvolvido pelo NotebookLM, abordei sobre a doutrina clássica de Benjamin Graham ligando às operações financeiras nacionais.
Além de abordar questões mais avançadas sobre investimento, o alvo também são para as pessoas que queiram iniciar investimento para que tenham mais segurança nesta operação.
Além de abordar questões operacionais, Graham enfatiza a questão emocional e a consistência, compenentes ideal a ser considerado principalmente para quem está iniciando.
O objetivo deste Projeto é mostrar e reforçar aos brasileiros que investimento é uma ferramenta de autonomia. Aprender a linguagem do dinheiro, você deixa de ser "refém" de um sistema e passa a fazer o dinheiro trabalhar por você.
